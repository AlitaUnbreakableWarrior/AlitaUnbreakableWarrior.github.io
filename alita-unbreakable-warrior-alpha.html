Alita: Unbreakable Warrior
-
Alpha

Version 0.4.5

After a Concept Discovery and two Vertical Slice, this Sunday we have released the alpha of Alita: Unbreakable Warrior. Although the build is still not very stable, this is the first version that shows the direction the game is going in. The main features that contains, including the ones from the previous versions, are listed below. We will write blog posts explaining how we implemented some of them, so stay tuned for further information.

Player:

- Alita basic controller

- Movement (pathfinding)

- Triple attack (combo)

- Dash

- First skill

- Second skill

- VFX

- FX

- Lose condition

Map:

- Narrative dialogs

- First level props placement

- Breakable chests

AI:

- Netman melee

- Behaviour

- VFX

- FX

- Netman range

- Behaviour

- VFX

- FX

- Barrelman

- Behaviour

- VFX

- FX

- Enemies loot

Graphics:

- Cartoon shader

- Decals

- Particle systems

- Lights

UI:

- Basic HUD

- Alita inventory

- Skill tree (non-functional)

- End screen

Audio:

- Interactive music

- FX
