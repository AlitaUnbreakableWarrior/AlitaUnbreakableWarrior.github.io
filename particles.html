Particles!

A tour to our particle system

To make the game more interesting and visually appealing, we have added a particle system to our JellyBitEngine. This particle system allows us to create beautiful particle effects.
We can modify the particle emitter properties, such as the speed at which the particles moves, the acceleration of the particles, the rotation of the particle when it appears and the speed at which it rotates, the lifetime of the particles, the size, etc.

We can also make or particles appear in different shapes, this shapes are box, sphere and cone.

Box

Sphere

Cone

This way we can create very interesting and varied effects, while improving the player's inmersion.



Another very useful tool is changing the color of the particle.



We can decide whether a solid color or a gradient over time.



Adding this to the possibility of applying animated textures greatly improves the quality of the effects that we can generate
